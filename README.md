# apps-grandlisboa-resource

## 项目简介

本仓库用于归档和发布多个独立 HTML 页面资源。  
这些页面可以用于信息展示、工具界面、静态原型等多种场景。  
仓库内容不针对任何单个域名或具体网站，仅作为通用 HTML 资源集合使用。

## 目录结构

```
.
├── pages/           # 存放独立的 HTML 页面文件
├── assets/          # 公共资源（样式、脚本、图片等）
├── docs/            # 文档或说明文件（可选）
└── README.md        # 本文件
```

- `pages/`：每个 HTML 文件为一个独立页面，可直接在浏览器打开或部署到任意静态托管服务。
- `assets/`：页面引用的公共资源，按需组织。
- `docs/`：存放相关文档、使用说明或版本记录。

## 页面归档说明

本仓库中的 HTML 页面均为独立归档，彼此之间无强依赖关系。  
每个页面均包含完整的 HTML 结构，部分页面可能引用 `assets/` 下的公共资源。  
页面内容可能包含示例数据、模拟界面或功能性组件，均不指向真实业务或特定服务。

## 使用方式

1. 克隆或下载本仓库：
   ```bash
   git clone https://github.com/your-username/apps-grandlisboa-resource.git
   ```
2. 直接打开 `pages/` 下的 HTML 文件即可查看效果。
3. 也可将 `pages/` 目录部署到任意静态服务器，通过 URL 访问对应页面。

## 维护说明

- 页面更新：新增或修改页面时，请确保文件放置在 `pages/` 目录下，并更新相关资源引用。
- 资源管理：公共资源请统一放入 `assets/`，避免重复。
- 版本记录：建议在 `docs/` 中维护变更日志，方便追踪历史。
- 兼容性：页面尽可能兼容主流浏览器，但部分特性可能需要现代浏览器支持。
